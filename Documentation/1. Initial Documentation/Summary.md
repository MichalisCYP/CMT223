# Problems we want to solve

1.  Short attention span

2.  Digital distractions

3.  Poor time management

4.  Lack of structured productivity

5.  Inefficient study/work environments

6.  Burnout and mental fatigue

# Features we want to implement and which problem they solve

- Real-time study/work environment monitoring (5):
  - Sensors: light, loudness, temperature and humidity

- Focus estimation (1,2):
  - Eye tracking

  - Presence tracking

  - Phone detection (post-MVP)

  - Head pose (looking away detection and distance sensor)

  - Desk monitoring (post-MVP)

  - Inactivity monitoring

- Study Session Management (3,4,6):
  - Schedule study sessions

  - Pomodoro timer during sessions

  - Focus estimation during sessions

  - Break recommendation

- Feedback Intervention -- Physical Interaction (4,5)
  - Gentle alerts according to the above monitoring data

  - Physical feedback (OLED screens)

  - Physical interaction (buttons, potentiometer)

- Planning and Analytics Platform (web first, mobile post-MVP) (4)
  - Session scheduling

  - Session summaries

  - Environmental data
