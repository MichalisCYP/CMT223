# Dashboard UX

## MVP Screens

- Live session
- Session history
- Alerts panel (can live inside main page)

## Live Session (Priority)

Must show:

- session title and state
- timer + current phase
- focus score
- current sensor values
- presence/CV status
- latest alerts
- actions: start, pause, resume, finish, acknowledge

Suggested layout:

- top: timer + focus + state
- middle: sensor cards
- bottom: event feed + controls

## History

Show:

- past sessions
- average focus per session
- simple trend charts (focus/environment)
- break recommendations vs acknowledgements

## UX Rules

- readable at a glance
- no debug noise on main screen
- actionable alerts only
- keep demo path obvious
